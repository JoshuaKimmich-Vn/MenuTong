# Healthcare Management System

WinForms C# project using OOP-style separation:

- Models: data classes
- Services: calculation, planning, alert, reminder and record services
- Pages: UserControl pages for Home, Profile, Record, Water Reminder, Report
- UI: theme colors
- Form1: main navigation only

## How to open

Open `HealthcareManagementSystem.sln` in Visual Studio.

Target framework: `net8.0-windows`.

If your Visual Studio uses an older .NET version, edit `HealthcareManagementSystem.csproj` and change:

```xml
<TargetFramework>net8.0-windows</TargetFramework>
```

to your installed Windows target, for example:

```xml
<TargetFramework>net6.0-windows</TargetFramework>
```
